# 1주차 미션 - 숫자 야구 게임 ⚾ 

---

## ✍🏻 구현할 기능 목록

>  ### ⌨️ 세팅 (Mac 기준)
>
> - java 버전 확인 및 변경
> - 컨벤션에 맞추어 새줄 문자 LF 로 변경 



### 🖥  상대방(컴퓨터)

> - 서로 다른 임의의 수 3개 세팅



### 👩‍💻사용자

> - ### ⌨️ 입력 받기 
>
>   - 사용자 숫자일 경우
>     - 3자의 수인가?
>     - 1~9 범위 안에 있는 숫자인가?
>     - 서로 다른 수인가?
>   - 게임이 끝난 경우 
>     - 1과 2인가?
>   - 공통
>     - 숫자가 아닌 문자일 경우
>
>   ---
>
> - ### 🖥 출력 하기
>
>   - 입력 문구
>   - 힌트 문구
>   - 종료 문구
>   - 재시작/종료 설명 문구
>
>   ---
>
> - 게임 진행 로직
>
> - 정답 확인 로직
>
> - 힌트 출력 로직
